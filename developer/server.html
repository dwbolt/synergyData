The first step is to install node.js

  1. Install node Download newest LTS version (18.16.0 as of 05/05/23)
  2. node -v # in a terminal window to make sure node insrtalled
  3. Create a folder for your developemnt. Open a termianl session inside the folder you just created.
  4. git clone https://github.com/dwbolt/synergyRoot.git # download local server
  5. cd synergyRoot
  6. ls -al # to see if clone worked
  7. ./scripts/_1-Init-synergyRoot #init git submodules
  8. ./scripts/_3-HTTP #start server on port 80
  9. test your local server in your browser
