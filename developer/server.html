The first step is to install node.js

  1. Install node Download newest LTS version (18.16.0 as of 05/05/23)
  2. Create a folder for your developemnt.
  3. git clone https://github.com/dwbolt/synergyRoot.git # download local server
  4. ./scripts/_1-Init-synergyRoot #init git submodules
  5. ./scripts/_3-HTTP #start server on port 80
  6. test your local server in your browser
